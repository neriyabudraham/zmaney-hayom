# זמני היום - Zmaney HaYom

אפליקציית Android לזמני היום (זמני תפילה) עם שעון מעורר והתראות.

## תכונות

- **זמני היום** - חישוב מדויק של 15 זמנים הלכתיים (עלות, הנץ, סוף שמע, חצות, שקיעה, צאת הכוכבים ועוד)
- **שעון חי** - שעון דיגיטלי עם תאריך עברי וגרגוריאני
- **התראות חכמות** - הגדרת התראות לכל זמן, עם אפשרות להתראה X דקות לפני/אחרי
- **ניהול התראות** - ממשק אחיד לניהול כל ההתראות
- **היסטוריית התראות** - צפייה בכל ההתראות שהיו
- **שעון קיץ/חורף** - אוטומטי או ידני
- **ערכת נושא** - מצב בהיר וכהה
- **מיקומים** - ירושלים, תל אביב, חיפה, באר שבע, או מיקום מותאם אישית

## דרישות טכניות

- **Android 4.4 (API 19)** ומעלה
- תמיכה במסכים קטנים (240x320)
- מותאם ל-Qween 1 ומכשירים דומים

## טכנולוגיות

- Java + Android SDK
- KosherJava Zmanim Library (חישובי זמנים הלכתיים)
- Material Design Components
- SQLite (אחסון התראות והיסטוריה)
- AlarmManager (התראות מדויקות)

## בנייה

```bash
./gradlew assembleDebug
```

ה-APK יהיה ב: `app/build/outputs/apk/debug/app-debug.apk`
